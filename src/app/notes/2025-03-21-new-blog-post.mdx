## Markdown Features

### Text Formatting
You can use **bold** or *italic* text for emphasis. You can also use `inline code` for technical terms.

### Lists
Unordered lists:
- First item
- Second item
- Third item with **bold text**

Numbered lists:
1. First point
2. Second point
3. Third point
